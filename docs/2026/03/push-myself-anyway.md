# 趁着年轻，我偏要勉强--詹青云

## 总结

1. 这本书在讲什么（1句话）：作者从贵州厂矿小学，到香港中文大学，一路走到哈佛法学院，走入律师圈，因为辩论走入了娱乐圈。记录了作者考试学习、读书、旅行的人生感悟。
2. 哪3点让我印象最深:
   1. 作者从小对读书的热爱，以及对喜欢的事情的坚持；在书中引经据典，旁征博引可见作者读书之丰富。
   2. 作者在旅行途中遇到的人，书写的事都如此有趣。
   3. 作者能够从学习的过程中感受到快乐与满足，而不是只有最后的成功才获得快乐；因此能够坚持做很多事而不放弃，毕竟结果总是要三五个月，甚至三五年才能有所收获。

## 笔记

> “芥川龙之介三十岁那年做了一道算术题：想想自己就算还能高效阅读三十年，每年保持五十本书的状态，这一生，也不过还能再读一千五百本书，他大哭，没有想到人生是这么有限。”  

> “千山万水，随意行去”，永远不要低估自己可以走到的地方。趁着年轻，我偏要勉强。”

## 感想

从作者与父母的相处中，看到家庭对孩子的影响是巨大的。在每一次走向更高学府的时候，都感受到自己与别人的差距，但作者都能凭借自己的努力成为其中优秀的人，适宜变化，不惧改变。作者对旅行的态度，与对人的包容，以及能够以有趣的眼光看待世事，让人看起来很开心，是我最新的部分。

无论是读书还是旅行，因人而异，因看书的不同年龄而异。世俗的成功你也许一辈子也无法达到，不要去羡慕远方的，别人的人生，过好当下的每一天。
